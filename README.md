# Calculadora de Combustão Industrial (BoilerCalc)

Uma aplicação web moderna para cálculos de parâmetros de combustão em caldeiras industriais.

## Funcionalidades

- Cálculo de PCI (Poder Calorífico Inferior)
- Determinação de Calor Útil e Entrada de Energia
- Cálculo de Relação Ar/Combustível (Teórica e Real)
- Estimativa de Vazão de Ar (Massa e Volume)
- Suporte a múltiplos combustíveis (Lenha, Bagaço, Carvão, Óleo)
- Composição de combustível personalizada

## Tecnologias

- Python 3
- Flask
- NumPy
- HTML5 / CSS3 (Design Responsivo)

## Como Usar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute a aplicação:
   ```bash
   python app.py
   ```

3. Acesse no navegador:
   `http://localhost:5000`
